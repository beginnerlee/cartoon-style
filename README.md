I write a code to switch the image to a cartoon format.
This python code is a code that converts a image into a cartoon style. It was implemented using the opencv functions of thresholding, binary filter, and media blur. If you put a dolphin.png picture in the cartoon.py code and run it, you can get the same result as cartonstyle.png. In addition, if you put a picture of pb.png and run it, you can get the result like cartonstyle2.png. However, if you look at the results of carton style2.png, you can see a limitation that only the polar bear picture and the nearby environment are applied to the caron style, but not to the overall(Ex.mountain).
